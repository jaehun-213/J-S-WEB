## 🎯 주요 기능

### 1. 회사소개 (About Us)
- CEO 인사말
- 회사 연혁 (타임라인)
- 보유 면허 및 인증
- 조직도
- 기술인력 현황
- 오시는 길 (본사/지사, 지도 연동)

### 2. 사업영역 (Business Area)
- 금속구조물/창호/온실공사업
- 지붕판금/실내건축공사업
- 제조업 (모듈러 제작)

### 3. 시공실적 (Portfolio)
- 8개 이상의 실제 시공 실적 갤러리
- 카테고리별 필터 기능
- 프로젝트 상세 정보

### 4. 고객지원 (Support)
- 공지사항 게시판
- 온라인 견적 문의 폼
  - 파일 첨부 기능 (최대 3개, 각 10MB 이하)
  - 개인정보 수집 및 이용 동의
  - 실시간 폼 검증

## 🎨 디자인 시스템

### 컬러 팔레트
- **Primary Color (Burgundy)**: `#6a1a2b` (메인 컬러)
- **Primary Color Dark**: `#4d1320`
- **Primary Color Light**: `#8b2d42`
- **White**: `#FFFFFF`
- **Gray Scale**: `#F3F4F6` ~ `#111827`

### 타이포그래피
- **폰트**: Malgun Gothic (맑은 고딕)
- **기본 크기**: 16px
- **반응형 폰트 크기**: 모바일 환경 최적화

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - CSS 변수 (Custom Properties)
  - Flexbox & Grid Layout
  - 반응형 디자인 (Mobile First)
- **JavaScript (Vanilla)**:
  - DOM 조작
  - 이벤트 처리
  - 폼 검증
  - 이미지 슬라이더
  - 스무스 스크롤

## 📱 반응형 디자인

- **Desktop**: 1200px 이상
- **Tablet**: 768px ~ 1024px
- **Mobile**: 768px 이하
  - 모바일 햄버거 메뉴
  - 터치 친화적 버튼
  - 최적화된 폰트 크기

## 🚀 사용 방법

1. 모든 파일을 웹 서버에 업로드합니다.
2. 브라우저에서 `index.html`을 엽니다.
3. 각 페이지 간 네비게이션을 통해 탐색합니다.

### 파일 구조
```
제이앤에스 홈페이지/
├── index.html          # 메인 페이지
├── about.html          # 회사소개
├── business.html       # 사업영역
├── portfolio.html      # 시공실적
├── support.html        # 고객지원
├── styles.css          # 공통 스타일시트
├── script.js           # 공통 JavaScript
└── README.md           # 프로젝트 문서
```
